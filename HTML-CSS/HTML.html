ENEBA CLONE

Este es un clon de la página de Eneba, una plataforma de venta de videojuegos digitales. El objetivo de este proyecto es practicar y mejorar mis habilidades en HTML y CSS, creando una réplica lo más fiel posible al diseño original de Eneba.

En este proyecto, he utilizado HTML para estructurar el contenido de la página, creando elementos como encabezados, párrafos, imágenes y enlaces. Además, he aplicado CSS para darle estilo a la página, utilizando colores, fuentes y diseños similares a los de Eneba.

El resultado final es una página web que se asemeja mucho a la original de Eneba, con una apariencia atractiva y funcional. A través de este proyecto, he aprendido a trabajar con HTML y CSS de manera más efectiva, mejorando mis habilidades en el desarrollo web.
